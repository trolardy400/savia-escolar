# Savia Escolar 🍃 — Gestión Contable con Raíz y Propósito

[cite_start]**Savia Escolar** es una plataforma SaaS innovadora diseñada para transformar la manera en que los centros educativos, directivas de cursos y asociaciones de padres de familia administran sus eventos y recaudaciones de fondos de manera colaborativa y transparente[cite: 1, 20].

## 🎯 ¿Qué Problema Resolvemos?

[cite_start]La plataforma combate los cuellos de botella clásicos en la administración escolar[cite: 2]:
* **Falta de Transparencia:** Los apoderados desconocen a menudo cómo se gestionan los fondos; [cite_start]Savia ofrece visibilidad integral[cite: 4, 5].
* [cite_start]**Seguimiento de Morosidad:** Elimina el caos del seguimiento manual de deudas mediante un monitor de deudores críticos[cite: 6, 11].
* [cite_start]**Reportes Desorganizados:** Sustituye excels y libretas por un panel centralizado con métricas en tiempo real[cite: 7, 8].

## ✨ Funcionalidades Principales

1. [cite_start]**Dashboard Inteligente 📊:** Visualización de métricas globales, total recaudado y alertas de morosidad automatizadas[cite: 9, 11].
2. [cite_start]**Gestión de Eventos 📅:** Creación de campañas con metas económicas, plazos definidos y división en múltiples cuotas[cite: 12, 13].
3. [cite_start]**Control de Estudiantes 👥:** Perfiles individuales que gestionan un ecosistema de pagos por grado y contacto[cite: 14].
4. [cite_start]**Transparencia Total 🧾:** Exportación a PDF (jsPDF) e integración revolucionaria con **Google Sheets** para rendición de cuentas pública[cite: 16, 17].
5. [cite_start]**Multidivisas y Personalización 🌍:** Soporte para más de 7 divisas y temas Light/Dark optimizados para la lectura[cite: 19].

## 🎨 Sistema de Diseño (UI/UX)

El diseño sigue una filosofía de **"Gestión con propósito"**, utilizando tonos botánicos para reducir el estrés de manejar dinero escolar:

* **Tipografía:** *Playfair Display* (Serif) para un estatus institucional Premium e *Inter* (Sans) para máxima legibilidad en la interfaz.
* **Paleta de Colores:** * **Gama Bosque:** Verdes saturados para acciones primarias y confianza.
    * **Gama Salvia:** Tonos tierra para fondos y superficies neutras.
* **Componentes:** Uso extensivo de *Glassmorphism* (.glass-card) y microinteracciones con Framer Motion.

## 💻 Stack Tecnológico

[cite_start]Savia Escolar está construida sobre tecnologías de punta para asegurar un rendimiento instantáneo[cite: 20]:
* [cite_start]**Frontend:** React v19 + Vite[cite: 21].
* [cite_start]**Backend:** Firebase (Firestore para base de datos reactiva y Google Auth)[cite: 22].
* [cite_start]**Estilos:** Tailwind CSS v4 + Framer Motion[cite: 23].
* [cite_start]**Automatización:** Google Apps Script para sincronización con hojas de cálculo[cite: 17, 24].
